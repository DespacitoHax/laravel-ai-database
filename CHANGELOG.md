# Changelog

All notable changes to `laravel-ai-database` will be documented in this file.

### Added
- Initial release
- Support for Anthropic Claude, OpenAI, and Google Gemini
- DB facade methods: `ask()`, `askForQuery()`, `askDetailed()`
- Artisan command: `db:ask`
- Strict mode for security
- Schema caching
- Smart table detection
- Multi-database support (MySQL, PostgreSQL, SQLite)
